# Habit Tracker

A web app that helps users maintain and improve their habits using spaced-repetition reminder.

Technologies used:

-  [React](https://reactjs.org/)
-  [TailwindCSS](https://tailwindcss.com/)
-  [Firebase](https://firebase.google.com/)
